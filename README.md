# 💼 TalentHub

A modern **Talent Management & Recruitment Platform** built with **Python** and **Flask**. TalentHub streamlines the hiring process by connecting recruiters with talented candidates through an intuitive web interface. The platform enables job posting, candidate management, application tracking, and efficient recruitment workflows for organizations of all sizes.

> **Connecting talent with opportunities through a smart recruitment platform.**

# ⚙️ Installation

## 1. Clone the Repository

```bash
git clone https://github.com/subham-paul/TalentHub.git
```

```bash
cd TalentHub
```

---

## 2. Create Virtual Environment

### Windows

```bash
python -m venv venv
```

Activate

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv venv
```

Activate

```bash
source venv/bin/activate
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4. Configure Environment Variables *(Optional)*

Create a `.env` file if required.

```env
SECRET_KEY=your_secret_key
DATABASE_URI=your_database_connection
```

---

## 5. Run the Application

```bash
python app.py
```

or

```bash
flask run
```

# 🔄 Workflow

1. Users register or log in.
2. Recruiters create and publish job postings.
3. Candidates browse available jobs.
4. Candidates submit applications and upload resumes.
5. Recruiters review applications and shortlist candidates.
6. The platform manages the recruitment process efficiently.
